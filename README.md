
# Skipli AI - Front-end

Front-end for Skipli AI web app 


## Run Locally

Clone the project

```bash
  git clone https://github.com/ngotrongphuc/skipli-ai-frontend
```

Go to the project directory

```bash
  cd skipli-ai-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
